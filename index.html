Journey Friction

Transition points are always the hardest.

For elderly travellers, airport friction often appears at transition points, the moments where they need to decode what happens next.

Opportunity Statement

How might Etihad help older travellers navigate international transfers with greater confidence?

Ideas

Designing more confident connections.

01

Transfer Preview

A personalised transfer guide shared after booking, with large visuals, simple language, and clear step-by-step guidance, helping first-time and older passengers prepare confidently before arriving in Abu Dhabi.

Solution image 01 Use file name solution-transfer-preview.jpg
Digital onboarding page or video explaining the transfer before travel.
02

Transfer Guide

>A transfer guide placed in every seat pocket, with clear illustrations and walking expectations, designed to be quickly understood by older passengers while remaining useful for all travellers.

Solution image 02 Use file name solution-transfer-guide.jpg
Printed guide placed in the seat pocket before landing.
03

Personal Transfer View

A QR code near departure screens that lets older passengers enter their flight number and access only what they need: gate details, walking directions, boarding status, and nearby help points.

Solution image 03 Use file name solution-personal-transfer-view.jpg
Passenger scanning QR code and viewing personalised transfer information.
